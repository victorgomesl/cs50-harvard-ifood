# Exercício 4: Crédito (desafio)

Este programa valida números de cartão de crédito usando o algoritmo de Luhn e identifica se o cartão é American Express, MasterCard, Visa ou inválido.

O programa pede ao usuário o número do cartão de crédito e, em seguida, informa se o número é válido e a qual bandeira o cartão pertence.

## Como Executar

Compile o programa com make credit.

Execute o programa com ./credit e insira o número do cartão de crédito quando solicitado.
